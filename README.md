## ✨ Features

- **Admin Login** – secure login to enable edit mode.  
- **Business Details** – display and edit logo, name, address, phone, and owners (admin only).  
- **Services Management** – add, view, and manage services (admin only).  
- **Meetings Management** – schedule and view meetings.  
- **Responsive UI** – built with Material UI components.  
- **State Management** – MobX for global state.  

---

## 🛠 Technologies

- **Frontend:** React, React Router v6, Material UI, SweetAlert2  
- **State Management:** MobX  
- **API:** Fetch API

## 📖 Usage

### Admin Users

- Log in to access **edit mode**.  
- Edit business details (**logo, name, address, phone, owners**).  
- Add and manage **services**.  
- Manage **meetings**.  

### Regular Users

- View **services**.  
- Schedule **meetings**.
